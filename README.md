# Prompt

**STILL A WORK IN PROGRESS**

Easily build interactive CLI's in Elixir. 

## Installation

Add `prompt` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:prompt, "~> 0.1.0"}
  ]
end
```
